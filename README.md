## 使い方
1. SGraphを継承して好きなグラフのクラスを作る
2. 自由に実験

基本的にVisualStudioを推奨。理由は
* パスが通してあるのでそのまま使える
* デバッガが優秀
* Intellisenseによる入力補助が便利
* そもそも他環境では試していない
などなど。

仕様例をWikiのチュートリアルに書いたのでそれを見るとわかるかも。
